# @wissem_hajbi/pi-nointl

Pi package for scanning Next.js projects for untranslated strings with `nointl`.

## Install

```bash
pi install npm:@wissem_hajbi/pi-nointl
```

## Usage

- Slash command: `/nointl-scan [path]`
- Tool: `scan_untranslated_strings`

The package depends on `@wissem_hajbi/nointl` and resolves it locally, so it is portable across machines once installed through pi.
